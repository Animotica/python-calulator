This is a calculator for windows. So there is no guaranty that it works on other OS.

How to Run the Calculator:
To run the Calculator open the file derectory where your Calculator is in and double click it.

How to use the first mode of the calculator?
Just enter 1
Than enter the first number and than the second. It will calculate:

1. number 1 + number 2
2. number 1 - number 2
3. number 1 * number 2
4. number 1 / number 2
